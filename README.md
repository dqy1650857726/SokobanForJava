# SokobanForJava
----
## 介绍
### java写的简单推箱子游戏，可自定义关卡，游戏自动保存。
### maven构建,控制台输入java -jar tuixiangzi.jar可直接运行
### 在map目录下可自定义关卡,  新建txt文件(关卡数.txt)
//        4-墙
//        0-空白
//        1-玩家
//        2-箱子
//        3-目标位置
### 例子：
##### 4,4,4,4,4,4,4,4,4
##### 4,0,0,0,0,1,0,0,4
##### 4,0,2,0,0,3,0,0,4
##### 4,0,0,0,0,0,4,0,4
##### 4,4,4,4,0,0,0,0,4
##### 4,0,0,0,0,0,4,4,4
##### 4,0,0,0,0,0,0,0,4
##### 4,0,0,0,0,0,0,0,4
##### 4,4,4,4,4,4,4,4,4
### 可自己跟换游戏素材(游戏用到的图标都在img里,替换时保持文件名不变即可)
